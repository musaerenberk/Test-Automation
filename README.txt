This project is testing the pages of the e-commerce site:
opens the home page of the site,
registers within the site,
searches within the site,
provides a tabular display of products,
provides a list display of products,
examines the searched product on the site,
shows the product examined in a large visual,
adds the selected product to the wishlist,
adds the selected product to the cart,
removes the product added to the cart,

It checks the name of the website's and provides information about it.
It also takes a screenshot of each function and creates a file with the date.
It gives information about the success or failure status of each function after its completion.
If you want, you can view the test result as a report in Browsers. For this you need to do: test-output > html > index.html > open in browser
